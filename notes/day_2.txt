======

1. python -m venv name_of_venv --> to create venv

ex: python -m venv my_venv

2. name_of_venv\scripts\activate  --> to activate venv

ex: my_venv\scripts\activate

3. pip install dep_name --> to install dep
	ex: pip install pandas

4. pip freeze --> list all packages installed in sys/env

requirements.txt:
=================
* to store dep for a project

5. pip freeze > requirements.txt --> take all module store in requirements.txt file

6. pip install -r requirements.txt --> it will install all modules or dep present in requirements.txt file.
